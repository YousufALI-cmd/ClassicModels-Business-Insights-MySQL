# ClassicModels-Business-Insights-MySQL
SQL analysis of ClassicModels database to uncover sales trends, customer behavior, and top-performing products using advanced queries.

## 📌 Overview

SQL-based analysis of the ClassicModels database to extract actionable insights on customer behavior, sales performance, and regional demand patterns.

## 🛠 Tools Used

* MySQL

## 🔍 Key Analyses

* Identified top-selling products based on revenue and order quantity
* Analyzed country-wise demand distribution
* Evaluated customer order frequency and patterns
* Applied advanced SQL concepts: JOINs, subqueries, aggregations, and window functions
